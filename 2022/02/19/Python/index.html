# 前言（Python 3）

Python 是一种高级通用脚本编程语言。解释执行。

# Python 的特性

  • 高级脚本语言
    类似 shell 脚本，可交互地单步直译运行，但又不只是脚本（处理复杂任务、面向对象 OOP）
  • 胶水语言
    可以将不同语言（C/C++、Java）编写的程序 “粘” 在一起的语言
  • 丰富的数据结构和库
    内建数据结构（list、dict 等），强大易用的标准库和扩展库

# 行与缩进

Python 中可以使用缩进来表示代码块，从而不需要使用大括号 =={ }==。

  • 同一个代码块的语句必须包含相同的缩进。
# 正确样例
if True:
    print ("True")
else:
    print ("False")

# 错误样例
if True:
    print ("Answer")
    print ("True")
else:
    print ("Answer")
  print ("False")# 缩进不一致，会导致运行错误
