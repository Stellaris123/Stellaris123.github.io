# DP 主要步骤

# （1）维数判断。

包含一定属性的 N 维集合。

  • 集合：包含所有选法且：①只考虑前 j 个物品。②总体积小于等于 i。

  • 属性：max，min，数量

# （2）状态转移

集合划分。

将不同维度的，划分为不同的子集

# 背包问题

有 N 个物品和一个容量为 V 的背包。求在背包容量不超的情况下选择总价值最多的物品。

# 01 背包

  • 额外条件：每种物品只能用一次。

# 维数判断

维度：当前选择物品，当前容量。

集合条件为：①只考虑前 i 个物品。②总体积小于等于 j。

属性：最大总价值。

# 状态转移

两种状态：①不含第 iii 个物品。②含第 iii 个物品。

①从 111 到 i−1i-1i−1 之间选且总体积小于等于 jjj，即 f(i−1,j)f(i-1,j)f(i−1,j)。

②从 111 到 iii 之间选且总体积小于等于 jjj，即 f(i−1,j−vi)+wif(i-1,j-v_i)+w_if(i−1,j−vi​)+wi​。

其中②只在 j≥vij\ge v_ij≥vi​ 时成立。

  • 状态转移方程：f(i,j)=max(f(i−1,j),f(i−1,j−vi)+wi)f(i,j)=max(f(i-1,j),f(i-1,j-v_i)+w_i)f(i,j)=max(f(i−1,j),f(i−1,j−vi​)+wi​)。
