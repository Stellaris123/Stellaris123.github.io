# 一、快速排序（分治）

①确定分界点：x=q[l]，q[r]，q[l+r>>1]x=q[l]，q[r]，q[l+r>>1]x=q[l]，q[r]，q[l+r>>1]。

②调整区间：一边小于等于xxx，一边大于等于xxx。

③递归处理左右两边（注意左右两边的边界）

伪代码：iii 指针从数组左边界的左边开始，jjj 指针从数组右边界的右边开始。x 采用l,rl,rl,r 中间的值。先移动iii 指针，当iii 指针所指的数大于xxx 的时候停止并开始移动jjj 指针，当jjj 指针所指的数小于 x 的时候停止。此时若 i<j，则进行交换。

性质：任意时刻，i 左边的数始终小于等于 x，j 右边的数始终大于等于 x。当两指针相遇的时候结束。

时间复杂度：平均O(nlogn)O(nlogn)O(nlogn)，最坏O(n2)O(n^2)O(n2)。

# 整数域二分

  • 注意终止边界、左右区间取舍。

（1）缩小范围：r=midr=midr=mid，l=mid+1l=mid+1l=mid+1，取中间值：mid=(l+r)>>1mid=(l+r)>>1mid=(l+r)>>1。

（2）缩小范围：l=midl=midl=mid，r=mid+rr=mid+rr=mid+r，取中间值：mid=(l+r+1)>>1mid=(l+r+1)>>1mid=(l+r+1)>>1。

第一种不会取到rrr 这个值，第二种不会取到lll。可以用来处理无解的情况。最初的二分区间从[1,n][1,n][1,n] 扩大到[1,n+1][1,n+1][1,n+1] 和[0,n][0,n][0,n]，若最后二分终止在这个越界的下标上，则无解。

  • 右移运算向下取整，整数除法向零取整。
